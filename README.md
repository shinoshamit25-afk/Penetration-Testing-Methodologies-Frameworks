# Penetration Testing Methodologies Frameworks
## Overview

This repository documents my study of industry-recognized penetration testing methodologies, security assessment frameworks, and adversary behavior models.

The goal of this repository is to demonstrate practical knowledge of:

- Cyber Kill Chain
- Penetration Testing Principles
- OSSTMM
- OWASP WSTG
- NIST SP 800-115
- PTES
- ISSAF
- MITRE ATT&CK
- Industry-Specific Security Frameworks
